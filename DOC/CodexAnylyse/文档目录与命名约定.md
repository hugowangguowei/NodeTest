# 文档目录与命名约定

## 1. 验收类文档目录

自本约定生效起，以下文档统一放入：

- `DOC/CodexAnylyse/验收/`

适用类型：

- 人工验收清单
- 人工验收记录
- 验收结论记录

## 2. 验收类文档命名规则

验收类文档文件名必须带日期和时间，格式统一为：

- `YYYY-MM-DD-HHMMSS-人工验收清单.md`
- `YYYY-MM-DD-HHMMSS-人工验收记录.md`
- `YYYY-MM-DD-HHMMSS-验收结论记录.md`

说明：

- `YYYY-MM-DD-HHMMSS` 为 24 小时制时间戳
- 同一天允许存在多份验收文档
- 不再使用只有日期、没有时间的验收文件名

## 3. 当前执行规则

当前项目后续如新增以下文档，必须遵守本约定：

- 验收清单
- 验收记录
- 验收结论

非验收类文档暂不强制迁移到该目录，也不强制全部带时间。
