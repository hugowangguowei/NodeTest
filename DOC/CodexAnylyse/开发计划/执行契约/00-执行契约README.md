## 1. 目录用途

本目录用于存放当前项目的本地执行契约单。

当前项目已经接入外部 Tracker，但本目录仍承担“本地最小可执行任务合同补充层”的作用。

## 2. 当前规则

1. 执行契约应能脱离聊天记录独立理解
2. 执行契约应引用对应的 WBS 节点
3. 执行契约进入 `已自测` 或 `待人工验收` 时，必须回链：
   - 自测报告
   - 验收清单
   - 会话交接
