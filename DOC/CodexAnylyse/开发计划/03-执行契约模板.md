# 执行契约模板

更新时间：2026-03-29

## 1. 用途

当后续把工作切成可执行子任务时，执行契约应至少满足本模板。

该模板可用于：

- Issue
- Task
- 本地执行单
- 外部协作系统中的任务卡片

## 2. 模板

```md
目标：
- ...

Owner：
- ...

Write Scope：
- ...

Depends On：
- ...

验收入口：
- 主入口：...
- 辅助入口：...

验收标准：
- ...

输出物：
- ...

证据链接：
- 计划文档：...
- 设计文档：...
- 自测报告：...
- 验收清单：...
- 交接记录：...

当前状态：
- 待开发 / 开发中 / 已自测 / 待人工验收 / 已验收 / 阻塞中

上级节点：
- ...
```

## 3. 当前项目执行规则

在当前项目未接入外部 Tracker 前，如需新增细粒度任务，优先在本地采用该模板，再决定是否同步到外部工具。
