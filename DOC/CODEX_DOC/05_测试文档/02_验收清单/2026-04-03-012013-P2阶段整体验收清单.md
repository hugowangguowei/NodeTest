# 人工验收清单 - P2阶段整体能力（仓库模型/工厂编排/统一回写）

时间：2026-04-03 01:20:13

## 1. 验收目标

验证 P2 阶段“知识仓库状态域与对象模型 -> 软件工厂异构对象编排 -> 统一回写”链路整体可用。

## 2. 验收步骤

1. 启动服务并打开 `http://127.0.0.1:3000/`
2. 在页面依次执行：
   - `生成场景`
   - `生成表达`
   - `记录构建流程`
   - `执行工厂编排`
   - `执行统一回写`
3. 切换右侧输出 Tab，确认可见：
   - `软件定义`
   - `工作台表达`
   - `构建流程`
   - `工厂编排`
   - `回写结果`
4. 验证工厂接口：
   - `GET /api/factory/catalog`
   - `POST /api/factory/compatibility/check`
   - `POST /api/factory/assembly/plan`
   - `POST /api/factory/assembly/execute`
   - `POST /api/factory/publish`
5. 验证回写接口：
   - `POST /api/repository/writeback`
   - `GET /api/repository/writeback/logs?limit=5`
   - `GET /api/repository/writeback/snapshot`
6. 执行自动化测试：`npm test`
